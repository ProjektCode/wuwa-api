# Changelog

All notable changes to the WuwaAPI project will be documented in this file.

## [1.0.0] - 2026-02-07

### Added

*   Initial release with Fastify REST API.
*   Assets-driven importer for characters and weapons.
*   New character fields added: `combatRoles`, `officialIntroduction`, `releaseDate`, `nation`, `gender`, `class`, `birthplace`, `additionalTitles`, `affiliations`, `skinImages`, `resonanceChain`.
*   Aemeath character data added.
*   Basic CORS allowlist, security headers, and rate limiting implemented.
*   `meta` endpoint with dataset hash.
